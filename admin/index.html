How to edit your library

  1. Sign in with a GitHub token (not “Sign in with GitHub”, unless OAuth is set up).
  2. Open Albums, then click one album — you edit one album at a time.
  3. Expand a track to change audio, paste lyrics, or upload a .lrc / .vtt file.
  4. Click Publish. Wait a minute for GitHub Pages, then refresh the public site.
Lyrics tips

Prefer uploading a premade .lrc or .vtt file. Or paste all lines into Lyrics text — one line per lyric. Timed example: [0:12] Hello there
